# Tocksup

Tocksup is a library inspired by yowsup but aims to avoid the 'bad_baram' error by adapting to the latest WhatsApp protocols.

## Features

- [ ] Send and receive messages
- [ ] Handle multimedia messages
- [ ] Group management
- [ ] Avoid 'bad_baram' error
